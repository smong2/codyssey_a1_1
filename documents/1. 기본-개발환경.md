* git config 
```

* user.name, user.email -> 아래와 같이 입력

git config --global user.name="smong2"
git config --global user.email="smong2@gmail.com" 

```

* python3 출력

```
python3 -c 'print("Hello Python!")'
Hello Python!

-c option 으로 파일을 만들지 않고 코드를 즉석으로 실행
```
