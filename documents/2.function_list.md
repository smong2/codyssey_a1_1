## 프로그램에 사용한 함수 리스트

1. 화면 제어 및 공통 유틸리티 (UI/UX 담당) clear_screen(): 화면을 쾌적하게 유지하기 위해 이전 출력물들을 지워주는 함수입니다. 윈도우(cls)와 Mac/Linux(clear) 환경을 모두 지원합니다.

- **pause_screen():**

  사용자가 처리 결과(예: 저장 완료, 에러 메시지)를 읽을 수 있도록 엔터 키를 칠 때까지 화면을 잠깐 멈춰주는 역할입니다.

- **get_valid_input(prompt_msg):**

  사용자가 실수로 엔터만 치거나 띄어쓰기만 입력했을 때, 프로그램이 튕기거나 오류가 나는 것을 막고 계속 올바른 입력을 요구하는 방어용 함수입니다.

- **get_multiline_input(prompt_msg):**

  프롬프트 본문처럼 여러 줄의 텍스트를 입력받을 때 사용합니다. :q를 누르면 저장을, :c를 누르면 취소를 담당하여 사용자 편의성을 높였습니다.

2. 터미널 한글 정렬 도구 (레트로 감성의 핵심)

- **get_display_width(text):**

  터미널에서 영어는 1칸, 한글은 2칸을 차지합니다. 이 차이를 계산하기 위해 글자를 옛날 방식인 euc-kr로 변환하여 실제 화면에서 차지하는 너비(바이트 길이)를 정확히 계산해 냅니다.

- **pad_string(text, total_width, align):**

  위에서 구한 너비를 바탕으로, 표의 칸 크기에 맞춰 남는 공간을 띄어쓰기(공백)로 채워줍니다. (왼쪽, 오른쪽, 가운데 정렬 지원)

3. 데이터 가공 및 리스트 출력 로직 (핵심 엔진)

- **get_filtered_prompts(...):**

  이 프로그램의 핵심 필터입니다. 카테고리 일치 여부, 검색어 포함 여부, 즐겨찾기 여부, 조회수 정렬 여부를 파라미터로 받아서 조건에 맞는 데이터만 리스트로 쏙 뽑아주는 역할을 합니다.

- **print_prompt_list(target_list, title):**

  전달받은 데이터 리스트를 PC 통신 감성의 예쁜 표 형태로 화면에 그려주는 '시각화' 담당 함수입니다.

- **show_list_and_detail(fetch_func, title):**

  리스트를 보여준 후, 사용자가 글 번호를 선택하면 상세 보기(detail_prompt)로 넘겨주는 징검다리 역할을 합니다. 상세 보기에서 수정/삭제를 하고 돌아오면 목록을 최신 상태로 새로고침합니다.

4. 자료실 기능 (데이터 영속화)

- **save_prompts():**

  메모리에 떠 있는 현재 프롬프트 상태를 data 폴더에 .json 파일로 영구 저장합니다. 덮어쓰기 경고 기능이 포함되어 있습니다.

- **load_prompts():**

  저장된 .json 파일을 읽어서 현재 프로그램으로 덮어씌웁니다. (이전에 작업하던 상태로 복구)

- **export_to_markdown():** 다른 사람들과 공유하거나 깃허브 등에 올리기 좋게, 현재 프롬프트들을 카테고리별로 잘 묶어서 .md 파일로 변환하여 출력해 줍니다.

5. 주요 비즈니스 로직 (CRUD)

- **add_prompt():**

  신규 프롬프트를 등록하는 과정을 담당합니다. 현재 존재하는 카테고리를 분석해서 1, 2, 3번 메뉴로 보여주고, 사용자가 원하면 0번을 눌러 새 카테고리를 만들 수도 있게 설계된 매우 똑똑한 함수입니다.

- **detail_prompt(prompt_obj):** 특정 프롬프트를 선택했을 때 실행됩니다. 진입하자마자 조회수(views)를 1 올려주고, 전체 내용을 보여줍니다. 하단 메뉴를 통해 제목/내용 수정, 리스트에서 삭제, 즐겨찾기 온/오프 기능을 수행할 수 있습니다.

6. 서브 메뉴 라우터

- **run_submenu(title, options, actions):**

  서브 메뉴들이 공통적으로 가지는 "메뉴 그리기 -> 입력받기 -> 뒤로 가기 처리" 로직을 하나로 묶어둔 템플릿 함수입니다.

- **manage_prompt_menu(), manage_category_menu(), manage_favorite_menu():**

  각각 '프롬프트 게시판', '카테고리 분류함', '즐겨찾기 보관함'에 들어갔을 때 어떤 메뉴 항목을 띄우고 어떤 함수를 실행시킬지(actions) 매핑해 주는 세팅 함수들입니다.

7. 메인 진입점

- **show_main_menu():**

  프로그램 켰을 때 가장 먼저 보이는 대문 화면을 그려줍니다.

- **main():**

  프로그램의 심장입니다. 무한 루프(while True)를 돌면서 사용자의 입력을 기다리고, 입력값(match choice)에 따라 6번의 서브 메뉴들이나 저장/종료 기능으로 길을 안내해 줍니다.
